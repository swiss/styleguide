Pagination

FLEX

The paging function allows the user to browse through a large number of elements which are arranged into pages. The element appears above and below the elements list.

Object - 2 of 3
  • Previous
  • 1
  • 2
  • 3
  • Next
<nav class="pagination-container">
  <span>Object - 2 of 3</span>
  <ul class="pull-right pagination">
    <li class="separator"><a href="#"><span class="icon icon--step-backward"></span></a></li>
    <li class="separator"><a href="#">Previous</a></li>
    <li><a href="#">1</a></li>
    <li class="active"><a href="#">2</a></li>
    <li><a href="#">3</a></li>
    <li class="separator-left"><a href="#">Next</a></li>
    <li class="separator-left"><a href="#"><span class="icon icon--step-forward"></span></a></li>
  </ul>
</nav>

Anchor links

FLEX

An anchor link is a jump label which allows the user to navigate to a particular position within the page currently in use.

  • Link 1
  • Link 2
  • Link 3
<ul>
  <li class="anchor"><a class=" icon icon--before icon--root" href="#">Link 1</a></li>
  <li class="anchor"><a class=" icon icon--before icon--root" href="#">Link 2</a></li>
  <li class="anchor"><a class=" icon icon--before icon--root" href="#">Link 3</a></li>
</ul>

Process navigation

FLEX

Process navigation is used in connection with forms that are spread over several pages. The active process stage is marked in red. The previous steps are clickable and lead the user back to the selected process stage.

  • Step 1
  • Step 2
  • Step 3
  • Step 4 
<nav class="nav-process">
  <ul>
    <li>
      <a href="#">Step 1</a>
    </li>
    <li>
      <a href="#">Step 2</a>
    </li>
    <li class="active">
      <a href="#">Step 3</a>
    </li>
    <li class="disabled">
      <a href="#">Step 4</a>
    </li>
  </ul>
</nav>

Tab Navigation

Tab navigation is used to group related contents locally within the same page. By way of exception, the font color of the tabs is dark gray (rather than blue) here. The active tab is highlighted with the red bar and the white background color.

Please note that the contents in the tabs are only used for presentation purposes here. The actual contents are contingent on the purpose of their use (e.g. search results or further information).

Reference in detailed specification: Chapter 2.5.4
